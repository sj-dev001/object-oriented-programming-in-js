# object-oriented-programming-in-js

A simple shopping cart project built with Object-Oriented JavaScript and the DOM.

## Overview

This project recreates a shopping cart using an object-oriented approach. It demonstrates how
`Product`, `ShoppingCartItem`, and `ShoppingCart` work together to manage products, quantities,
and cart totals.

## Features

- `Product` class for product data
- `ShoppingCartItem` class for cart item data and item total calculation
- `ShoppingCart` class for managing cart items
- Add products to the cart
- Increase, decrease, and remove cart items
- Live cart totals and quantity summary

## Project Structure

- `index.html` — page markup
- `css/styles.css` — responsive UI styling
- `js/script.js` — OOJ classes and DOM logic

## How to Run

Open `index.html` in a browser.

## How to Test

1. Add a product to the cart.
2. Add the same product again to increase its quantity.
3. Use the plus and minus buttons to adjust quantity.
4. Remove an item from the cart.
5. Clear the entire cart.
