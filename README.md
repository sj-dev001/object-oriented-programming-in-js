# object-oriented-programming-in-js

A simple shopping cart project built with Object-Oriented JavaScript and the DOM.

## Features

- `Product` class for product data
- `ShoppingCartItem` class for cart item data and item total calculation
- `ShoppingCart` class for managing cart items
- Add products to the cart
- Increase, decrease, and remove cart items
- Live cart totals and quantity summary

## Project Structure

- `index.html` — page markup
- `css/styles.css` — responsive UI styling
- `js/script.js` — OOJ classes and DOM logic

## How to Run

Open `index.html` in a browser.
